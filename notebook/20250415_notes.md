# Genomic Offset Analyses MVP Oysters

Starting genomic offset analyses for the oysters included in the MVP Chesapeake Bay experiment.

## Overview

I will be using gradientForest, LEA, and RDA to calculate offset. I will be using seascape data to train models, then experimental individuals to validate these models. The goal here is to make predictions about how each of 8 wild populations and 2 selection lines will perform in Chesapeake Bay conditions, then determine how well these predictions line up with experimentally obtained fitness metrics. 

So far I have pipelines for LEA and GF that are tailored to another project and which need to be updated for this project, but I still need to build a pipeline for RDA. I am waiting still on the complete seascape genomic dataset for training, but have the environmental data needed for training and some of the environmental data needed to predict to the Chesapeake environment during the experimental period.
