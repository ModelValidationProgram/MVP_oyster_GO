## Data for "Genomic Offsets Predict Survival with Low Accuracy in a Marine Common Garden"

This folder contains all the data used in the present study. 

EnvDat contains all environmental data and is delineated into experimental and seascape data. Data that applies to both is hosted in the main EnvDat folder. This includes the unprocessed data for the experimental environment, but not raw files for the seascape sites or experimental populations, as these data were taken from a separate seascape study. Other data include the scaled experimental data and environmental distance matrices. Seascape data used for model training and model prediction, scaled an unscaled, can be found in the seascape folder.

ExpDat contains all fitness and metadata for the experiment, including raw length and survival data, and information about the bags, sites, and populations included in the experiment.

GenDat contains the genotype matrices used for model training, both pre- and post-processing.

IndDat contains information about the individuals in both the seascape data and the common garden experiment.
